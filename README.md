# GitPulse

A GitHub profile analyzer that fetches your repos, events and stats,
scores your activity and generates a fully customized README.md —
ready to download and drop into your profile repo.

## How it works

1. Enter any GitHub username in the search form
2. A loading screen shows live progress while the app fetches your profile, repos and events simultaneously from the GitHub API
3. Your analysis dashboard loads — profile card, stats, language breakdown, repo quality scores, collaboration badge, activity insights and recent event feed
4. Switch to the README Generator tab — confirm your tech stack, fill in optional fields like role, LinkedIn and a fun fact
5. Preview the generated README in both rendered and raw markdown views
6. Download it as `README.md` and drop it straight into your GitHub profile repo
